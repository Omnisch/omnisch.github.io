文献考试一句话总结

发表于 2025-10-26 分类于 luon

粗体为两个方向共有的文献。

0872

  • 共产党宣言

    资产阶级产生了无产者，消灭私有制，消除阶级

  • 人类简史

    认知革命、想象建构、性本冷漠贪婪罪恶

  • 信息简史

    发展史—理论体系—影响，非洲鼓语-理论与实际结合，汉字符号含义丰富-高语境，码本-真学问，数理逻辑学-敢于质疑

  • 大数据时代

    核心就是预测。放弃样本/接受全局，放弃精确/接受混乱，放弃因果/接受相关

  • 佐藤大：用设计解决问题

    あったらいいな，谁也没见过/谁也不需要，模糊扫视，交流—表达 (善于比喻)，一定能被解决的问题

  • 设计心理学 4

    D.诺曼，现在是半自动化，机器未来要有本能—行为—反思层，示能 (无声说明书)，自然信号/可预测性/持续感知/自然映射

  • 为真实的世界设计

    现在什么样/应该怎么样，一切都是设计，最根本的工作是改造人类的环境和工具，社会道德责任感，废弃-异化，回归自然，人是通才

  • 设计与文化导论

    现代主义—后现代主义1940，炫耀性的消费，身份认同，设计是技术与文化之间的桥梁

  • 技术与文明

    始生—古生—新生代，代偿 (浪漫主义/地方主义)，战争是机械化社会的戏剧

  • 美的历程

    【周】实践理性【汉】人对自然的征服【魏晋】为艺术而艺术【唐】活力/律令【宋】伤感矛盾无我之境【元】有我之境【明清】浪漫感伤批判现实

  • 心流：最优体验心理学

    探讨怎样才能快乐幸福。
    注意力有限，让精神有序就是进入心流。
    有目标、有反馈、有挑战。

  • 游戏的人

    定义：自愿性、独立性/非功利性、规则性。
    游戏可以是严肃的。

  • 游戏改变世界

    优秀的游戏机制 移植到 现实议题。
    幸福的高效/紧迫的乐观/社交织网/史诗意义

  • 游戏设计艺术

    以玩家体验为出发点和核心。
    生长图：设计师—过程—游戏—玩家—体验

  • 体验引擎：游戏设计全景探秘

    见《心流》。
    体验引擎—游戏制作—流程

  • 社交网络改变世界

    Web 2.0 革命
    ISP：身份、地位、权力
    3D：多元化、民主化、分散化

  • 人机交互之道

    定性研究 (众包研究/问卷调查)
    获取数据 (传感/注意力/日志)
    分析数据 (回溯分析/智能体建模/社会网络分析)

  • 奇点临近

    加速回报定律，指数增长
    GNR：信息与生物/信息与物理/强人工智能

  • 微积分的力量

    无穷，阿基米德方法，数学领域发现而非发明，微积分用于各领域，牛顿vs莱布尼茨

  • 机器学习

    线性模型、支持向量机、决策树、神经网络、集成学习、特征选择、稀疏学习、强化学习

  • 研究是一门艺术

    转换到读者的视角思考。
    提出问题、解决问题
    论文写作流程：观点、理由、证据、回应、论据

  • 科研成功之道：像外行一样思考，像专家一样实践

    大胆假设，小心求证。
    科研想法：KISS 理论 (复杂问题简单化)
    科研效率：计算机辅助
    表达结果：重点优先、内容分布、悬疑小说。

  • Qualitative inquiry and research design

    定性研究的理论基础和实践应用
    非数字化的数据
    叙事研究、现象学研究、案例研究、以理论为基础的研究、批判研究

  • 世界现代设计史

    见《为真实的世界设计》《设计与文化导论》。
    19 ~ 21 世纪的工业设计
    以人为本、KISS 原则

  • 艺术与视知觉

    理性分析美学表现：画面平衡、形状、颜色、光线……
    格式塔，颜色的象征意义

  • 认知与设计：理解 UI 设计准则

    见《艺术与视知觉》。
    减轻认知负担。

  • 交互设计：超越人机交互

    见《游戏设计艺术》《认知与设计：理解 UI 设计准则》。
    三种交互方式：认知交互/社会交互/情感交互
    交互方式展望

  • 用户界面设计

    见《游戏设计艺术》。
    可靠性、安全性、标准化、可移植性、易学习性、高效性
    交互风格：直接操纵/命令行/自然语言

  • 认知心理学

    研究人类思维过程的科学
    感知 (感觉/知觉)、注意 (分散/集中)、记忆、语言、推理 (演绎/归纳)

  • 游戏设计梦工厂

    见《心流》《游戏设计艺术》。

  • Games user research

    见《游戏设计艺术》《人机交互之道》。
    用用户研究的方法论理解玩家，针对需求构建游戏体验。

  • 游戏感

    实时操控 (提供反馈)、模拟空间 (主动感知/物理交互)、润色 (提升交互效果)

  • Designing game feel. A survey

    见《游戏感》。另，三种游戏感设计元素：Physicality, Amplification, Support

  • 走出教育游戏的迷思

    旨在通过游戏活动促进学习的工具，能培养知识/技能/思维，从科学证据研究增值/结果/横向比较

  • 语境提供者：媒体艺术含义之条件

    媒体世界中的艺术家是语境提供者
    艺术作品共同创作，可以代表群体意愿。

  • Understanding the lives of problem gamers

    半结构式访谈。
    游戏有意义和目的：生活的一部分
    推拉因素：个人/人际/环境

  • Tangible user interfaces

    将数字信息和物理实体相结合，用实体对象表示信息
    不同领域的应用：可视化、音乐、编程
    优缺点

  • Esports research: A literature review

    2020，关键词指导搜索，一竞技游戏二体育，语料库构建

  • 可视化与人工智能交叉研究综述

    可视化的人工智能 (数据质量改善/可解释) vs 人工智能+可视化 (特征提取/智能交互)

  • 混合现实中的人机交互综述

    真实-虚拟坐标轴，虚实融合/实时交互/三维注册，必须考虑人因，交互技术、存在的问题

  • Immersed in media telepresence theory…

    媒体中的遥现现象，即“临场感”。
    模拟人类感官体验：HMD、立体声、追踪识别……

  • 未来传媒: 我们的思考与教育的责任

    四个时代：传统媒体—融媒体—智能融媒体—智能媒体，三种形态：泛媒体、沉浸媒体、智媒体

  • 游戏性写实主义的诞生：动物化的后现代

    当代御宅族文学评论，思考社会与故事的关系。
    轻小说：角色、想象
    Galgame：循环、meta

  • 认知盈余

    极大推动社会发展的社会工程学，维基/开源
    违背经济学原理但精神层面的满足
    去中心化共享模式

  • 区分：判断力的社会批判

    翻译很烂，很难理解。
    人的思维习惯、文化水平被社会结构决定。
    批判康德的“无功利审美”思想。
